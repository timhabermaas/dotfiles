# Setup (Mac OS X)

1. Install ghostty
2. Install homebrew
3. Install `git` through homebrew
4. Install `fish` through homebrew and set as default shell: 
  1. Add fish path to `/etc/shells`
  2. `chsh -s /opt/homebrew/bin/fish`
5. Install `mise` through homebrew
6. Run `link.sh` to create symlinks from the different files in ~/.config to this repository
